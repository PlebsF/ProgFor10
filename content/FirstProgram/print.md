### printで、もじをひょうじしよう！ <a id="chapter-1"></a>
それじゃあ、printの使い方をくわしくみていくよ！
``` Python
print("ひょうじしたい文字")
```
のように、printでは""でかこまれたもじをひょうじできるよ！

ちなみに、
``` Python
print('ひょうじしたい文字')
```
のように、' 'でかこってもOK!

### printで、数をひょうじしよう！ <a id="chapter-2"></a>
かずをひょうじするには、
``` python
print(数)
# 例
print(10)
```
というようにすればいいよ！
