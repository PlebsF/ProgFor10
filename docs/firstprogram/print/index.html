文字をひょうじしよう！

もくじ

  • printで、もじをひょうじしよう！
  • printで、数をひょうじしよう！

前のページで少し言ったけど、print(プリント)は英語で「いんさつする」というえいごで、プログラミングでは「画面にひょうじしなさい」というめいれいだよ

日本語にプリンターがあるけど、これもプリントからきてるね！

printで、もじをひょうじしよう！

それじゃあ、printの使い方をくわしくみていくよ！

print("ひょうじしたい文字")

のように、printでは”“でかこまれたもじをひょうじできるよ！

ちなみに、

print('ひょうじしたい文字')

のように、’ ‘でかこってもOK!

printで、数をひょうじしよう！

かずをひょうじするには、

print(数)
# 例
print(10)

というようにすればいいよ！

じゃあじっさいにプログラムを書いてためしてみよう！ ここ
